Consulta #1 Obtener el listado de alumnos

SELECT * FROM ALUMNO

Consulta #2 Obtener el listado de cursos y las asignaturas asignadas

SELECT curso.nombreDescriptivo, asignatura.nombre FROM curso INNER JOIN asignatura ON curso.idCurso = asignatura.curso

Consulta #3 Obtener el listado de alumnos con beca

SELECT alumno.nombre, alumno.apellido1 FROM alumno WHERE alumno.beca LIKE 'si'

Consulta #4 Obtener listado de profes y su num.

SELECT profesor.nombre, profesor.apellido1, tlfcontactoprof.telefono FROM profesor INNER JOIN tlfcontactoprof

Consulta #5 Obtener listado de alumnos que esten en programación

SELECT * FROM alumno INNER JOIN asignatura WHERE asignatura.idAsignatura LIKE 'AS009'

Consulta #6 Obtener el listado de profes y sus asignaturas

SELECT profesor.nombre, asignatura.nombre FROM profesor INNER JOIN impartir ON profesor.idProfesor = impartir.idProfesor INNER JOIN asignatura ON impartir.idAsignatura=asignatura.idAsignatura

Consulta #7 Mostrar asignatura que tiene más alumnos y su total

SELECT asignatura.nombre, COUNT(matricula.idAlumno) FROM asignatura INNER JOIN matricula ON asignatura.idAsignatura = matricula.idAsignatura

Consulta #8 Obtener listado de alumnos que le de la clase el profe con ID PRO48 sin importar la asignatura

SELECT alumno.idAlumno, alumno.apellido1, alumno.nombre FROM alumno INNER JOIN matricula ON alumno.idAlumno=matricula.idAlumno INNER JOIN impartir ON matricula.idAsignatura=impartir.idAsignatura WHERE impartir.idProfesor='PR048'
